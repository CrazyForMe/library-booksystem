# 图书在线管理系统
###实现的一个简单的图书在线管理系统，使用的工具包括：
######1.ssm框架
######2.maven管理项目
######3.mysql数据库
######4.bootstrap前端框架
######5.myeclipse 2016
###使用方法：
######1.将项目导入到myeclipse中
######2.将book.sql文件导入到mysql中，导入语句为source 路径
######3.修改java/resources路径下的jdbc.properties文件，将数据库的账号和密码修改为自己的密码
######4.在浏览器中输入lhost:8080/BookSystem/
